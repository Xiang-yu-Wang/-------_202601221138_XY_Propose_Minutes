# 🎯 5 分鐘快速入門

## 給老闆的使用說明書

---

## 第一步：打開管理頁面

在瀏覽器網址列輸入：
```
http://localhost:5173/admin
```

> **提示**: 埠號 (5173) 可能會改變，請查看開發伺服器的輸出訊息

---

## 第二步：登入

使用 **Supabase 帳號** 登入：

1. 輸入您的 **Email**
2. 輸入您的 **密碼**
3. 點擊「登入管理後台」按鈕

> **沒有帳號？** 請聯繫系統管理員在 Supabase 後台建立帳號

![Login Screen]

---

## 第三步：新增公告

### 點擊「新增公告」按鈕

出現表單後，填寫以下資訊：

| 欄位 | 說明 | 範例 |
|------|------|------|
| **標題** | 公告的主要標題 | 春節期間服務通知 |
| **日期** | 公告發佈日期 | 2026-02-05 |
| **類型** | 選擇優先等級 | 重要 / 新消息 / 一般資訊 |
| **標籤** | 用逗號分隔，可不填 | 春節, 配送時間 |
| **內容** | 詳細的公告說明 | 春節期間照常收件，配送可能延後... |

### 點擊「新增」

✨ **完成！** 公告已立即出現在首頁！

---

## 第四步：檢查首頁

訪問首頁：
```
http://localhost:5173/
```

向下滑找到「最新公告」區域，應該能看到您剛新增的公告！

> **注意**: 使用 Supabase 雲端模式時，資料會即時同步，無需手動部署！

---

## 常用操作

### 編輯公告

1. 在管理後台找到要編輯的公告
2. 點擊「編輯」按鈕
3. 修改內容
4. 點擊「更新」

✨ **立即同步到首頁！**

### 刪除公告

1. 在管理後台找到要刪除的公告
2. 點擊「刪除」按鈕
3. 確認刪除

✨ **立即從首頁移除！**

### 備份公告

1. 點擊「導出」按鈕
2. 自動下載 JSON 檔案
3. 保存到安全的地方（Google Drive、Dropbox 等）

> **重要**: 請定期備份，防止資料遺失！

### 復原公告

1. 點擊「導入」按鈕
2. 選擇之前導出的 JSON 檔案
3. 確認導入

✨ **所有公告已恢復！**

## 公告類型說明

### 🔴 重要 (Important)
用於：緊急通知、時間敏感的信息
```
範例: 春節期間服務中斷通知
```

### 🟢 新消息 (New)
用於：新功能、方案調整、重要更新
```
範例: 2026 Q1 方案調整公告
```

### 🔵 一般資訊 (Info)
用於：普通通知、提醒、說明
```
範例: 上傳文件格式提醒
```

## 🚨 常見問題

### Q: 公告沒有出現在首頁？
**A:** 
1. 刷新頁面 (按 Ctrl+F5)
2. 確認是否點擊了「新增」
3. 檢查是否有錯誤訊息

### Q: 密碼忘記了？
**A:** 
聯絡開發者，他可以重置密碼

### Q: 能否在手機上編輯？
**A:** 
可以，但建議在電腦上編輯以獲得最佳體驗

### Q: 資料會保存在哪裡？
**A:** 
資料存儲在您的電腦瀏覽器中（localStorage），只有在此電腦上可見

### Q: 如何在其他電腦上編輯？
**A:** 
1. 先在此電腦導出備份
2. 在其他電腦登入管理後台
3. 導入備份

## 💡 使用技巧

### Tip 1: 定期備份
> 每次編輯後點擊「導出」備份，防止意外丟失資料

### Tip 2: 使用標籤
> 標籤幫助組織公告，方便日後查找

### Tip 3: 簡潔標題
> 標題要簡短清楚，讓用戶一眼看懂

### Tip 4: 詳細內容
> 內容要詳細完整，包含聯絡方式和相關連結

### Tip 5: 定期更新
> 定期新增公告，讓網站保持活躍
