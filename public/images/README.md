# 投票截圖示範圖片

請將實際的投票截圖放在此資料夾中：

## 需要的圖片

### 1. 電腦版投票截圖
- **檔案名稱**: `desktop-vote-example.jpg` (或 `.png`)
- **建議尺寸**: 寬 500-800px，高 400-600px（橫向）
- **需包含內容**:
  - 投票權憑證
  - 股東編號
  - 公司名稱
  - 投票內容

### 2. 手機版投票截圖
- **檔案名稱**: `mobile-vote-example.jpg` (或 `.png`)
- **建議尺寸**: 寬 350-500px，高 600-800px（直向）
- **需包含內容**:
  - 投票權憑證
  - 股東編號
  - 公司名稱
  - 投票內容

## 注意事項

1. **隱私保護**: 請確保截圖中的敏感資訊（如個人姓名、完整股東編號）已適當遮蔽
2. **圖片品質**: 確保文字清晰可讀
3. **檔案大小**: 建議壓縮至 500KB 以下，提升載入速度
4. **格式**: 支援 JPG、PNG、WebP

## 圖片壓縮工具

- [TinyPNG](https://tinypng.com/) - 線上壓縮
- [Squoosh](https://squoosh.app/) - Google 提供的壓縮工具
