## 🚀 如何使用（給老板）

### 第一次使用
1. 打開網站進入 `/admin` 管理後台
2. 點擊「交貨照管理」標籤（第三個，圖片圖示）
3. 點擊「新增交貨照」開始添加

### 每次添加新照片
1. 用 Imgur 等圖床上傳照片，取得圖片連結
2. 進入管理後台 → 交貨照管理
3. 點擊「新增交貨照」
4. 填寫：標題、日期、位置、圖片 URL、說明（選填）
5. 點擊「新增」完成
6. 前台網站會自動更新

## 💾 數據存儲方式

### localStorage 存儲
```typescript
// 存儲鍵名
const STORAGE_KEY = 'delivery_photos_data'

// 自動保存，無需手動操作
// 數據格式: JSON 數組
[
  {
    id: "1",
    url: "https://...",
    title: "台北地區交貨",
    description: "客戶滿意",
    date: "2026-01-29",
    location: "台北市"
  },
  ...
]
```

### 備份方案
- **導出**: 點擊「導出 JSON」下載備份檔案
- **導入**: 點擊「導入 JSON」恢復備份數據
- **重置**: 點擊「重置為預設」恢復初始狀態
