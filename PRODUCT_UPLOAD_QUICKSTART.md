# 🛍️ 產品上傳快速入門 (給老板)

## 簡單 3 步驟上傳商品

### 第一步:開啟管理後台
1. 打開瀏覽器,輸入網址後加上 `/admin`
   - 範例: `http://localhost:5173/admin`
2. 輸入密碼:`admin123`
3. 點擊登入

### 第二步:進入產品管理
1. 看到兩個頁籤:「公告管理」和「產品管理」
2. 點擊 **「產品管理」** (有包裹圖示)

### 第三步:新增商品
1. 點擊綠色按鈕 **「新增產品」**
2. 填寫資料:
   - 名稱:例如「高級禮盒組合」
   - 分類:例如「禮品盒」
   - 價格:450
   - 描述:簡單介紹產品
   - **上傳圖片** (最重要!)
3. 勾選「立即上架販售」
4. 點擊「新增」完成!

## 📸 圖片準備技巧

### 圖片太大怎麼辦?
如果系統提示「超過 500KB」:

**方法 1: 用手機拍照**
- 手機拍照通常已經壓縮過
- 直接上傳即可

**方法 2: 線上壓縮(推薦)**
1. 打開 https://tinypng.com
2. 把圖片拖進去
3. 等 3 秒自動壓縮
4. 下載壓縮後的圖片
5. 上傳到系統

**方法 3: 調整圖片尺寸**
- 用小畫家或預覽程式開啟
- 調整大小到 600x400 或更小
- 儲存後上傳

## ✏️ 修改商品

1. 找到要改的商品
2. 點擊「編輯」按鈕
3. 修改任何資料
4. 點擊「更新」儲存

## 🔄 上架/下架商品

- **上架**: 商品會在網站顯示,客人可以看到
- **下架**: 暫時隱藏,但資料還在

點擊產品卡片上的「上架」或「下架」按鈕即可切換

## 💾 記得備份!

**重要**: 資料只存在這台電腦的瀏覽器裡

每週記得:
1. 點擊「導出」按鈕
2. 存檔到電腦或雲端硬碟
3. 檔名會自動加上日期,例如:`products_2026-01-27.json`

換電腦時:
1. 在新電腦開啟管理後台
2. 點擊「導入」
3. 選擇之前導出的檔案
4. 所有產品就回來了!

## ❓ 遇到問題?

### 圖片上傳不了
→ 檢查圖片是否小於 500KB,用 TinyPNG 壓縮

### 商品沒出現在網站上
→ 檢查是否勾選「立即上架」,重新整理網頁試試

### 商品不見了
→ 可能清除了瀏覽器,用備份檔案匯入

### 忘記密碼
→ 聯繫技術人員修改
